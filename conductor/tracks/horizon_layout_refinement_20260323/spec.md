# Specification: Horizon Layout Refinement

## Overview
The goal is to refine the "Horizons of Focus" (Projects & Roles) view in the GUI application to be more visually appealing, better organized, and consistent with modern UI standards.

## Objectives
- [ ] Redesign the "Roles" and "Ambitions" (Projects) layout for better readability.
- [ ] Redesign the screen such that only "ambitions" are shown for the role that is selected.
- [ ] A click on a role should select it and show the related ambitions. 
- [ ] Another button should be added to edit the role
- [ ] Improve the "Project Details" pane to feel more integrated.
- [ ] Ensure consistent spacing and interactive feedback.

## Constraints
- Maintain existing functionality (CRUD operations).
- Adhere to the `product-guidelines.md`.
- No new external dependencies if possible.
